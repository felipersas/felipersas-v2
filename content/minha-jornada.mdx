# Minha Jornada como Desenvolvedor Full Stack 🚀

Olá! Quero compartilhar um pouco sobre minha trajetória no mundo do desenvolvimento e como cheguei onde estou hoje.

## O Início

Tudo começou no ensino médio, quando fiz um curso técnico de Desenvolvimento de Sistemas no SENAI. Foi lá que tive meu primeiro contato real com programação e percebi que aquilo era o que eu queria fazer da minha vida.

## Primeiros Passos

No começo, foi um pouco desafiador. Aprender a lógica de programação, entender como a web funciona, e tantas tecnologias diferentes... Parecia impossível às vezes. Mas com persistência e muita prática, tudo foi começando a fazer sentido.

### O que aprendi no início:

- **Lógica de programação** - A base de tudo
- **HTML e CSS** - A estrutura e aparência da web
- **JavaScript** - A vida da web
- **Git** - Controle de versão essencial

## Evoluindo

Depois de dominar o básico, comecei a me aprofundar em tecnologias mais modernas:

- **TypeScript** - JavaScript com type safety
- **React** - Biblioteca para criar interfaces
- **Next.js** - Framework React full-stack
- **NestJS** - Framework Node.js estruturado

## Stack Atual

Hoje trabalho principalmente com:

### Frontend
- Next.js (App Router)
- React 19
- TypeScript
- Tailwind CSS

### Backend
- NestJS
- Prisma ORM
- PostgreSQL / MySQL
- Docker

## O Futuro

Estou sempre aprendendo coisas novas. Atualmente estou focado em:

- **Machine Learning** - Explorando IA e ML
- **Rust** - Programação de sistemas
- **Arquitetura** - Melhorando design de software
- **DevOps** - CI/CD, cloud, infraestrutura

## Conclusão

A jornada como desenvolvedor é cheia de altos e baixos, mas é recompensadora. Se você está começando, não desista. Se você já está no caminho, continue aprendendo.

Até a próxima! 👋
