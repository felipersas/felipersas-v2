## Key Concepts

### Server Components

By default, components in the App Router are Server Components:

```tsx
// Server Component (default)
async function BlogPosts() {
  const posts = await fetchPosts()
  return <div>{/* ... */}</div>
}
```

### Client Components

Use `'use client'` for interactive components:

```tsx
'use client'

import { useState } from 'react'

export function Counter() {
  const [count, setCount] = useState(0)
  return <button onClick={() => setCount(c => c + 1)}>{count}</button>
}
```

### Data Fetching

Server Components make data fetching simple:

```tsx
async function UserProfile({ id }: { id: string }) {
  const user = await fetch(`https://api.example.com/users/${id}`).then(r => r.json())
  
  return (
    <div>
      <h1>{user.name}</h1>
      <p>{user.email}</p>
    </div>
  )
}
```

## Styling

### Tailwind CSS

Next.js pairs perfectly with Tailwind CSS:

```bash
npm install tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### CSS Modules

For component-scoped styling:

```tsx
import styles from './button.module.css'

export function Button() {
  return <button className={styles.btn}>Click me</button>
}
```
